Chapitre 2: Les réseaux informatiques

cette section a été faite par imad abdslam et rafik

  • Réseau informatique : ensemble d'ordinateurs et de terminaux autonomes connectés pour échanger des informations numériques dans un domaine géographique défini.
  • Mise en réseau : utilisation d'outils et de tâches pour relier les ordinateurs et permettre le partage des ressources.
  • Un réseau est composé d'équipements interconnectés permettant la communication de données entre applications, indépendamment des distances qui les séparent.
  • Évolution des réseaux : initialement utilisés pour la communication entre terminaux distants et un site central, ils intègrent désormais les données, la parole et la vidéo.

      • des notions fondamentales d'un réseau :

    1. Interconnexion : assure la transmission des données entre les nœuds.
    2. Communication : permet l'échange de données entre les processus (programmes en cours d'exécution).
    3. Nœud (node) : extrémité d'une connexion dans un réseau.
    4. Processus : programme en cours d'exécution, représentant le point de communication dans un réseau informatique.

2/ Les caractéristiques de base d’un réseau sont :

  1. .La topologie
  2. Le débit exprimé en bits/s (ou bps)
  3. La distance maximale (ou portée)
  4. Le nombre de nœuds maximum

3/ Quelques terminologies en réseau :

  1. Station de travail
  2. Noeud
  3. Paquet
  4. Réseaux Hétérogènes

Éléments d'un reseau :

  1. Les ordinateurs
  2. les logiciels
  3. Les supports du LAN
  4. Les équipements d'interconnexion du LAN

5/ Classification des réseaux :

Les réseaux peuvent être classifiés en fonction de différents critères :

  • Par portée l’envergure (La distance):

    1. LAN (Local Area Network = réseau local d'entreprise) : appelé réseau local, constitué d'ordinateurs et de périphériques reliés entre eux et implantés (planté) dans une même entreprise.
    2. MAN : (Métropolitan Area Network = Réseau métropolitain ou urbain) : Correspond à la réunion de plusieurs réseaux locaux (LAN)
    3. WAN : (Wide Area Network = réseau grande distance) : sont des réseaux étendus (large),et qui assurent la transmission des données sur des longues distances Il existe d’autres comme: PAN , WPAN , SAN

  • L’ouverture (leur utilisation) :

    1. Intranet : le réseau interne d’une entité organisationnelle ou d’une entreprise.
    2. Extranet : le réseau externe d’une entité organisationnelle ouvert vers l’extérieur.
    3. Internet Les réseaux publics

  • Les supports de connexion :

    1. Réseaux filaires
    2. Réseaux sans fils

  • Le type d’organisation :

    1. Les réseaux Postes à Postes (Peer to Peer, ou d’égal à égal)
    2. Les réseaux Clients/Serveurs: La plupart des stations sont des « postes clients »

6/ Le type de topologie :

  • Les réseaux en bus (ETHERNET, Token-Bus...) : Les machines sont reliées par un câble coaxial (le bus) , et chaque ordinateur est connecté en série sur le bus .
  • Les réseaux en étoile (téléphonie, STARLAN...) : Toutes les stations sont connectées à un commutateur, (un nœud central). SWITCH .
  • Les réseaux en anneau (FDDI, Token-Ring):: Les ordinateurs sont situés sur une boucle et communiquent chacun à leur tour. Elle utilise la méthode d'accès à "jeton" Token ring
  • Maillée (ATM, Réseaux MAN et WAN surtout) :Les réseaux maillés utilisent plusieurs chemins de transferts entre les différents nœuds.
