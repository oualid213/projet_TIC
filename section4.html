Introduction

  • Le Web et l'Internet c'est pareil

    Le World Wide Web (WWW) est un système hypertexte public sur Internet qui permet de consulter, avec un navigateur, des pages accessibles sur des sites.

    • Internet : support de communication général,
    • Web : une partie des contenus circulant sur l’Internet.

  • Principes du World Wide Web

    • Accéder à des documents textuels
    • Reliés entre eux par un mécanisme de lien « hypertexte »
    • De différentes natures : texte, image, son, vidéo, contenu applicatif...et interactif (l’achat/discussion....)

  • URL

    Un lien hypertexte est un lien qui permet de suivre une URL (ouvrir une nouvelle page) le nom unique de la page. décrivant l’emplacement d’une ressource. Elle contient :

    • L’adresse du serveur,
    • et l’adresse de la page sur le serveur.

Aspects techniques du Web

Les 3 mécanismes de base du Web sont :

  1. URL

    • Le Web permet d’accéder à un ensemble de ressources et la localisation peut faire appel au protocole DNS

  2. HTTP

    • Protocole de niveau applicatif, sans état et le paradigme client-serveur

  3. HTML

    • Langage de description de « pages Web »
    • Texte, images et autres objets
    • Programmation déclarative

Page web et site web :

Le web permet de consulter des pages accessibles sur des sites, décrites en utilisant des langages de description tels que HTML, XHTML ou HTML5 et stockées dans des fichiers .html ou .xhtml. Ces pages sont interprétées et affichées par un navigateur en utilisant une adresse URL et permettent d'accéder à d'autres pages en suivant des liens hypertextes. Un site web est un regroupement de pages autour d'une thématique commune, accessibles à partir d'une page d'accueil commune et structuré sous forme d'arborescence de dossiers et de fichiers. Les chemins relatifs peuvent être utilisés pour accéder à des fichiers situés dans des sous-dossiers

Création d’une page web HTML :

  • Un document HTML est :

    • Un fichier texte
    • Structuré par des balises
    • Interprété (visualisé) dans un navigateur
    • Créé dans un éditeur de pages web ou tout éditeur de texte

  • principes de langage HTML:

    les balises et principes de langage HTML

    Un fichier HTML doit comporter au minimum ces 4 balises :

    • Délimite le début et la fin du document :
      <html> ... </html>
    • Entête du document :
      <head> ... </head>
    • Titre du document : 
      <title> ... </title>
    • Corps du document : 
      <body> ... </body>
